How to Join

Follow these simple steps to start your adventure on MineVortex.

Java Edition

  1. Launch Minecraft: Java Edition (Version 1.21.6).
  2. Click on the "Multiplayer" button.
  3. Click "Add Server".
  4. Enter "MineVortex" as the Server Name.
  5. Enter minevortex.mine.bz as the Server Address.
  6. Click "Done" and join the server!

Bedrock Edition

  1. Launch Minecraft (Bedrock Edition).
  2. Click the "Play" button.
  3. Go to the "Servers" tab.
  4. Scroll down and click "Add Server".
  5. Enter "MineVortex" as the Server Name.
  6. Enter minevortex.mine.bz as the Server Address.
  7. The port should be 19132 (default).
  8. Click "Save" and join the server!
