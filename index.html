How to Join

Connecting to MineVortex is easy.

Java Edition

  1. Launch Minecraft: Java Edition.
  2. Click on "Multiplayer", then "Add Server".
  3. Enter MineVortex as the server name.
  4. Enter minevortex.mine.bz as the server address.
  5. Click "Done" and join the server!

Bedrock Edition

  1. Launch Minecraft (Windows 10, Mobile, Console).
  2. Click "Play", then navigate to the "Servers" tab.
  3. Scroll down and click "Add Server".
  4. Enter MineVortex as the server name.
  5. Address: minevortex.mine.bz
  6. Port: 19132 (default)
